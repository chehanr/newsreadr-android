# newsreadr-android

An android client for the [newsreadr-api](https://github.com/chehanr/newsreadr-api).

A work in progress.

![Screenshots](docs/img/screenshots.jpg)

## Usage

[Import](https://stackoverflow.com/questions/32197775/how-to-import-existing-java-project-into-android-studio?utm_medium=organic&utm_source=google_rich_qa&utm_campaign=google_rich_qa) the project to Android Studio or download a build from [here](https://drive.google.com/open?id=0B_Cm66eNqyNhZlI4aldTeDhwYlk) and install it.
